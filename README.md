# number-ai

A simple package that helps you work with numbers by using AI. You can use AI to discover patterns, do basic math, and make wise predictions.

## Installation

```bash
bun add number-ai
```

## Usage

```typescript
import { greet } from "number-ai";

console.log(greet("World")); // Hello, World!
```
